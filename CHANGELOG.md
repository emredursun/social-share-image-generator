# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-06

### Added
- Initial release of Social Share Image Generator
- Intelligent adaptive layout system for any dimension
- 6 professional layout templates (Auto, Centered, Split, Minimal, Bold, Elegant)
- Platform-specific presets (Facebook, Twitter, Instagram, YouTube, Story)
- Custom profile image upload functionality
- Customizable text fields (name, title, subtitle, website)
- Color customization for backgrounds and accents
- Text scale adjustment (50%-150%)
- 6 visual effects (Pattern, Circles, Grid, Waves, Dots, None)
- Export quality options (High, Ultra, Medium)
- Download as PNG functionality
- Copy to clipboard feature
- Real-time preview
- Responsive design for all screen sizes
- No dependencies - pure HTML5, CSS3, and JavaScript

### Features
- **Smart Layout**: Automatically adjusts to portrait, landscape, or square dimensions
- **Responsive Text**: Auto-scales based on canvas size
- **Professional Output**: High-quality images for all social platforms
- **Privacy-First**: All processing happens in the browser, no server uploads
- **Easy to Use**: No installation required, works offline after first load

### Technical Stack
- HTML5 Canvas API for image rendering
- Vanilla JavaScript (no framework dependencies)
- CSS3 with modern gradients and animations
- Plus Jakarta Sans font from Google Fonts
- Font Awesome icons
